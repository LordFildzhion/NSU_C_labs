# Heap sort

Realize heap sort

**Time for sorting - 3 seconds.**

## Input:

    Stirng №1: 
    Integer N in range from 0 to 2000000 - length of array

    String №2: 
    The array

## Output:

    Sorted array
    


## ```Example 1```

Input:
    
    3
    4 -1 2

Output:
    
    -1 2 4

## ```Example 2```

Input:

    3
    0 0 -1

Output:
    
    -1 0 0
